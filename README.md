# 🌱 EcoAssist AI

## Smart Waste Disposal Assistant for Visually Impaired People

### Problem Statement

Visually impaired individuals often face challenges in identifying and segregating waste correctly. Improper waste disposal affects recycling efficiency and environmental sustainability.

### Solution

EcoAssist AI uses computer vision, webcam-based object detection, and voice guidance to help users identify waste items and dispose of them in the correct waste bins.

### Features

1. Real-Time Object Detection using Webcam
2. Voice-Based Accessibility Guidance
3. Waste Classification

   * Plastic Waste
   * Paper Waste
   * Organic Waste
   * E-Waste
   * Metal Waste
   * Glass Waste
   * Reusable Items
4. Non-Waste Detection

   * Humans
   * Animals
   * Furniture
5. Accessibility Focused Design

### Technology Stack

1. React.js
2. TensorFlow.js
3. COCO-SSD Object Detection
4. React Webcam
5. JavaScript
6. HTML/CSS

### Workflow

1. User opens EcoAssist AI.
2. User starts webcam.
3. AI detects the object.
4. System classifies the item.
5. Voice guidance provides disposal instructions.
